# Tempus ⏱️

**Tempus** is a lightweight, browser-based time registration tool designed to make tracking work hours simple, fast, and frustration-free.

Instead of juggling spreadsheets or complicated systems, Tempus focuses on the essentials: quick time entry, clear weekly overviews, and reliable data storage.

---

## ✨ Features

- ⚡ Fast time entry with hours and notes  
- 📅 Clear weekly overview of registered time  
- 🔄 Instant switching between weeks  
- ➕ Easy row creation for new entries  
- 🧮 Automatic time totals  
- 💾 Local browser storage for reliability  
- 🎯 Simple, distraction-free interface
